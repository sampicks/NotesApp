## app module is based on following

- Clean Architecture
- Live Data with ViewModel
- Room Database
- Hilt
- Picasso
- Kotlin Coroutines
- View Binding
- Data Binding

- Live Data with ViewModel
- Retrofit
- Paging 3
- Navigation
